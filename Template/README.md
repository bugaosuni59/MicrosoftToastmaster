# 如何制作Agenda

- 1 下载模板 Agenda.xlsx
- 2 编辑Title 和 Member列
- 3 导出为图片
  - 1) Copy内容
  - 2) 右键-粘贴为图片 
    ![image](https://user-images.githubusercontent.com/24701101/177157270-97adf194-eafe-463e-a737-390a1f286054.png)
  - 3) 右键-Format Picutre: 设置背景为纯白
  - 4) 右键-Save as Picture

# 如何制作Poster

## 2) PPT Designer

- 2 编辑Title 和 日期
